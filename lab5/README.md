# Лабораторная работа №5

Лабораторная работа №5. Регулярные выражения: пять типовых задач на Pattern и Matcher.

## Описание

Изучить синтаксис регулярных выражений в Java, научиться использовать классы Pattern и Matcher для поиска и проверки совпадений в тексте.

## Запуск

Компилируется и запускается стандартными средствами JDK:

```text
javac *.java
java <ИмяКласса>
```
