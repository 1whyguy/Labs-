# Лабораторная работа №6

Лабораторная работа №6. Коллекции Java: List, Map, дженерики, потокобезопасные варианты.

## Описание

Познакомиться с Java Collections Framework, изучить основные интерфейсы (List, Set, Map), работу с обобщениями (Generics) и потокобезопасными коллекциями.

## Запуск

Компилируется и запускается стандартными средствами JDK:

```text
javac *.java
java <ИмяКласса>
```
