# Лабораторная работа №4

Лабораторная работа №4. Обработка исключений: стандартные конструкции, собственный класс исключения, логирование.

## Описание

Освоить механизмы обработки исключительных ситуаций в Java (try-catch-finally), научиться создавать собственные классы исключений и вести логирование.

## Запуск

Компилируется и запускается стандартными средствами JDK:

```text
javac *.java
java <ИмяКласса>
```
