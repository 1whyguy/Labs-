# Лабораторная работа №2

Лабораторная работа №2. Объектно-ориентированное программирование: иерархия классов, ООП-принципы.

## Описание

Изучить основные принципы ООП (инкапсуляция, наследование, полиморфизм) и реализовать иерархию классов на языке Java.

## Запуск

Компилируется и запускается стандартными средствами JDK:

```text
javac *.java
java <ИмяКласса>
```
