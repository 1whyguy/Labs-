# Лабораторная работа №7

Лабораторная работа №7. Многопоточное программирование: ExecutorService, Callable, CompletionService.

## Описание

Освоить основы многопоточного программирования в Java, научиться работать с пулами потоков (ExecutorService) и интерфейсом Callable для возврата результатов.

## Запуск

Компилируется и запускается стандартными средствами JDK:

```text
javac *.java
java <ИмяКласса>
```
