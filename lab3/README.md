# Лабораторная работа №3

Лабораторная работа №3. Хэш-таблицы: собственная реализация методом цепочек и применение HashMap.

Изучить структуру данных хэш-таблица, реализовать ее самостоятельно методом цепочек и применить стандартную коллекцию HashMap.

## Запуск
